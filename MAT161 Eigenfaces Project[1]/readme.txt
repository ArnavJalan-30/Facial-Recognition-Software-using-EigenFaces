Instructions for compilation:

1. Install Python, jupyter, matplotlib and numpy and setup environment
   The versions used are:I.	Python: 3.10.6
                         II.	Jupyter Notebook: 6.5.4
                         III.	Numpy: 1.24.3
                         IV.	MatPlotLib: 3.7
2. Open "Eigenfaces_for_recognition.ipynb" in a jupyter notebook with the Dataset in the same folder as the "Eigenfaces_for_recognition.ipynb" file.
3. Run all the cells
